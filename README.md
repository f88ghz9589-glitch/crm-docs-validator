# Валидатор документов CRM 2.0

Веб-приложение для проверки корректности документов абонентов Казактелеком перед загрузкой в CRM систему.

## Функционал

- **Проверка наличия всех обязательных документов:**
  - Бланк заказа
  - Заявление отказ от услуг
  - Доверенность
  - Удостоверение личности
  - Правоустанавливающие документы

- **Валидация форматов файлов:** PDF, JPG (макс. 10MB)

- **Проверка SMS кода подписи:** верификация через код, отправленный на номер абонента

- **Интеграция с CRM 2.0:** подготовка документов для загрузки

## Запуск

Просто откройте файл `index.html` в браузере. Никаких зависимостей не требуется.

## Технологии

- Чистый HTML5
- CSS3 (без фреймворков)
- Ванильный JavaScript
- SVG иконки
